# Trivia-TOP
Web page that makes questions about Twenty One Pilots, it uses the gemini API KEY to work

# Link deployment
link: https://gaeh-lpz.github.io/Trivia-TOP/

# Usage
Enter to the link above, a question about the band Twenty One Pilots, a band that i like a lot and stand for will be deployed, answer the question with one of the options
if the answer is correct you will be given a point in the other hand you will have an error, if you reach 3 wrong answer you lose your score and need to start again

# Questions
The questions are generated by Gemini by google, so if an answer is actually correct but the options said it wasn't it was not my fault but gemini

I hope you enjoy this project.
